Index.HTML dan CSS adalah dua bahasa yang berbeda namun saling melengkapi jika html kita analogikan
HTML adalah bahasa dasar untuk membuat struktur halaman web.
Dia menentukan apa saja yang muncul di halaman: teks, gambar, tautan, tabel, video, dsb. Kalau web itu rumah, maka HTML adalah rangka bangunan dan dindingnya.
Tanpa HTML, halaman web tidak akan punya isi seperti kerangka rumah dan CSS adalah bahasa untuk mengatur tampilan visual dari elemen HTML.
Dengan CSS, kita bisa mengubah warna, posisi, ukuran, font, bayangan, bahkan animasi.
jika HTML itu rangka rumah, maka CSS adalah cat, tirai, dan perabotan yang membuat rumah jadi indah.
Tanpa CSS, web terlihat polos dan membosankan.
